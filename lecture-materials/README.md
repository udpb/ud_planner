## 🎯 Design principles

The entire package reflects Underdogs education planning philosophy:

1. **Data flows top-down** — each step receives the previous step's output as input (RFP → Value Chain → assignments)
2. **Methodology is NOT one-size** — 9 methodologies (IMPACT · 로컬브랜드 · 글로컬 · 공모전설계 · 매칭 · 재창업 · 글로벌진출 · 소상공인성장 · 커스텀) selected per country
3. **One 5D per format** — a 36-hour bootcamp cannot install all 5 competencies; each country picks 2 (India picks Design as dominant + Direction as support)
4. **Execution-guaranteed** — every block ends with a tangible deliverable
5. **Quantitative saturation** — 520+ partners, 800 coaches, 4-axis rubric at 25% each, etc.

## 🧭 Assessment rubric (unified across all 5 countries)

| Axis | Weight | Definition |
|---|---|---|
| Idea specificity | 25% | Is the problem concrete? Is the insight non-obvious? |
| BM completeness | 25% | Who pays, how much, via what channel? Unit economics? |
| Growth potential | 25% | Can this 10x? What's the wedge + expansion path? |
| Impact | 25% | Market size × social/economic ripple |

**Jury**: 3 people = 1 public-sector expert + 1 private-sector expert + 1 Underdogs coach.

## ⚙️ A/B/C/D skeleton (all 5 assignments follow this)

- **A. Pre-event (2–3 weeks async)** — LMS videos, reading kit, individual exercise
- **B. Day 1** — Opening → Problem deep-dive → Team + ideation → MVP sprint #1 → Evening
- **C. Day 2** — Sprint #2 → Dry-run → Demo Day → Judging → Next-steps briefing
- **D. Post-event (2–8 weeks follow-up)** — Report, Stage 2 onboarding, alumni network
