# Part 3 — Casebook

> Reading this page, you will know the format every case follows and what to extract from each one.

## Why This Casebook Matters Most

This is the book's core. The fastest way for a new PM to build instinct is to read **many good cases, structured the same way**. Part 1 and Part 2 hold the abstract principles, but how those principles actually land in practice only becomes visible here.

The casebook keeps growing. Whether a win or a loss, every new project adds one entry, and the format is fixed so additions are frictionless.

## Shared Format — 6 Sections

Every case follows the same table of contents. When you move from one case to the next, the same information sits in the same place.

1. **One-line project summary** — funder · amount · duration · target · core scope
2. **Key message** — original text preserved (the slogan or declaration the proposing firm actually wrote)
3. **Impact Value Chain — 5 layers** — Input → Activity → Output → Outcome → Impact
4. **3 differentiation points** — the unique elements that set this bid apart from competitors
5. **How Underdogs' own tools and network were woven in** — how ACT-PRENEURSHIP, DOGS, IMPACT, LICORN, and others functioned inside this project
6. **What a new PM should learn from this case** *(most important)* — reusable building blocks, ways of thinking, cautions

## How to Read Each Case

- **Preserve the original message**: Section 2 slogans and declarations are expressions the proposing firm actually used. Copy-pasting them into another project strips the context and reads as off-key. Treat them as **references for building instinct**; when reusing, re-create them in the new project's context.
- **The 5 layers are a reference structure**: Actual proposals are rarely written in this 5-layer order. The guidebook restructures the content after the fact. The original proposals arranged sections in order of evaluation weight.
- **Differentiation means unique to context**: "Our firm is capable" is not differentiation. The phrase means "for this specific scope in this specific project, only this firm took this approach." It does not transfer verbatim to another project.
- **Verify Underdogs assets**: Read the figures and tool names against the current values in [Appendix B — Underdogs Assets Reference](../appendix/b-ud-assets.md). Numbers in the original case reflect the point in time when the proposal was written.
- **Linger longest on Section 6**: "What to learn from this case" is the true value of the casebook. You will gain more from this section than from the original-proposal summary.

## When a New Case Is Added

New cases are added through the guidebook agent brief at `.claude/agent-briefs/guidebook/add-case.md` (created when needed), following the same format. The goal is to add any project a new PM wins to the casebook within six months — the richest context lives in the memory of the person who just finished the project, and that is when documentation carries the highest asset value.
