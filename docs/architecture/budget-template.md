**구조**: 1-2 외부용보다 더 상세. **수익률 기준 (O열)** 까지 포함 → "어느 항목이 UD 수익으로 이체 가능한지" 판정.

핵심 컬럼:
| 컬럼 | 의미 |
|---|---|
| `B` | 항목 (인건비/시상금/교육운영비/후속관리/경상운영비) |
| `C` | 세목 |
| `D` | 세부내용 |
| `E` | 산출근거 |
| `F` | UD 예산 비목 |
| `G` | UD 수익 (O/X 표시) |
| `H` | 단가 |
| `I` | 명수/건수/시간 |
| `J` | 개월수/횟수 |
| `K` | 투입률 |
| `L` | 금액산출 (`=H*I*J*K`) |
| `M` | 비고 |
| `O` | 수익률 기준 (O/△/X) |
| `P` | 내용 (수익 가능 여부 코멘트) |

**최종 합계** (Rows 45~50):
- `L45` 사업비 계 = `=SUM(L22, L26, L41, L39, L44)`
- `L46` 일반관리비 (5%) = `=L45*5%`
- `L47` 기업이윤 (5%) = `=SUM(L45:L46)*5%` ← **외부용 1-2 의 8% 와 다름**
- `L48` 부가가치세 (10%)
- `L49` 총 사업예산
- `L50` 총 사업예산 (만원 미만 절삭) = `=ROUNDDOWN(SUM(L45:L48),-5)`

> **차이점**: 1-2 외부용은 일반관리비 5% + 기업이윤 8%. 2.내부용 세부는 5% + 5%. **양식별로 비율이 다름** → 우리 출력 시 각 양식의 고유 비율을 하드코딩.

## 6. 미해결·확인 필요

### 6.1 Q1 — 1-2 외부용의 운영진 인건비를 어떻게 우리 데이터에서 추출할까?

**현황**: 1-2 외부용은 운영진 인건비 (B11~D13) 를 "총괄 교육 관리자 7,500,000 × 6개월 × 0.1 투입률" 식으로 표기. 우리 시스템엔 PM/Director 의 월간 단가나 투입률이 없음.

**제안**:
- (A) `roleRateOverride` 필드에 직급별 월 단가 + 투입률 입력 폼 신설 → PM 수정 가능
- (B) 4-2 시트 B2G/B2B 단가를 코드 시드로 두고, ProjectType (B2G/B2B) 에 따라 자동 적용 + 투입률은 PM 입력
- (C) 외부용 출력 시점에 PM 이 모달로 입력 (비추천 — 매번 재입력)

→ **(B) 권장**. 4-2 단가는 옛 단가일 수 있으나 "현재 시점 디폴트"로 두고 PM 이 케이스별 수정.

**현황**: ContentAsset 모델에 `category=PRODUCT` 가 있고, 단가/할인 정보를 들고 있음 (Phase H Wave 4 이후).

**확인 필요**:
- DOGS / 비욘드에듀 / SV 임팩트 / 액션클럽 패키지가 ContentAsset 시드에 들어가 있는지
- 들어가 있다면 `4-3.언더독스 서비스 단가` 시트는 ContentAsset 의 read-only view 가 됨
- 들어가 있지 않다면 J2 Wave 에서 추가 시드

→ Phase J 진행 시 Phase H 시드 점검 필요.

### 6.3 Q3 — 일반관리비/기업이윤의 보정값 (-15,100) 원리는?

**현황**: 1-2 외부용 J26 = `=(J25+J24)*0.08-15100`. 보정값 15,100 은 단수 정리용으로 보임.

**확인 필요**:
- 22,000,000 케이스에서만 쓰이는 1회성 보정인지
- 모든 케이스에 적용되는 일반 룰인지 (예: 만원 단위로 떨어지게)

→ **재무팀 확인 필요**. 1차 구현 시엔 22,000,000 사례 그대로 보존 (수식 그대로) → 다른 케이스에서 검증 시 룰 도출.
