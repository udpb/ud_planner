# 코오롱 프로보노 — Value Chain 설계 교훈 케이스

> 이 케이스를 읽으면 **Value Chain 없이 장표부터 쓴 PM 이 어떻게 2주를 낭비하고, 어떻게 반나절 만에 방향을 잡았는지** 알게 됩니다. 가이드북에서 가장 강조하는 교훈의 원본.

## 1. 사업 한 줄 요약

- **발주처**: 코오롱 (대기업 B2B, 프로보노 · 임직원 재능 기부 프로그램)
- **사업명**: 코오롱 프로보노 프로그램 운영 (임직원이 사회적 기업·스타트업을 돕는 형태의 CSR 이벤트/교육 사업)
- **대상**: 코오롱 그룹 임직원 + 외부 소셜 섹터 창업가·단체
- **핵심 과업**: 임직원과 창업가를 매칭해 재능 기부·멘토링·협업 프로젝트 운영
- **기간**: 연·반기 단위

이 케이스는 수주 성과 자체보다 **사내 PM 업무 프로세스의 교훈** 이 더 유명합니다. 가이드북 Ch.5 (함정 1번) 의 실제 사례로 참조.

## 2. 핵심 메시지

원문 기조 요약:

> **"임직원의 전문성을 사회 문제 해결과 연결하는 프로보노 매칭 + 실행 중심 멘토링 플랫폼"**

B2B 프로보노 사업 특성상 "임직원 참여 만족도" 와 "사회적 임팩트" 의 두 축을 모두 잡아야 합니다. 만족도만 높고 임팩트 약하면 "이벤트로 끝난다", 임팩트만 강조하고 참여 경험이 약하면 "재참여율이 낮다" — 양쪽 배점이 다 있습니다.

## 3. Impact Value Chain 5계층

- **Input**: 코오롱 임직원 멘토 풀 · 언더독스 전담 PM · 소셜 섹터 창업가·단체 DB · 매칭 플랫폼 · 자체 방법론 (IMPACT·ACT-PRENEURSHIP) · 4중 지원 체계
- **Activity**: 임직원 멘토 모집·진단 → 창업가·단체 니즈 수집 → 매칭 → OT + 프로젝트 킥오프 → 주간 멘토링 (정해진 회차) → Action Week (현장 적용) → 결과 발표 · 성과 공유
- **Output**: 매칭 페어 수, 완료 프로젝트 수, 임직원 참여 시간, 창업가 산출물 수
- **Outcome**: 임직원 참여 만족도, 창업가·단체의 역량 변화 (진단 기반), 실제 사업 진도 변화, 재참여 의사
- **Impact**: 코오롱 CSR 포트폴리오 내 프로보노 프로그램 정착, 소셜 섹터 네트워크 확장, 임직원의 사회적 가치 연대감 누적

## 4. 차별화 포인트 3개

1. **매칭 품질 관리** — 임직원·창업가를 랜덤 매칭이 아니라 **사전 진단 기반 페어링**. 임직원의 전문성 영역과 창업가의 니즈를 ACT-PRENEURSHIP 5가지 역량 매트릭스로 맞춰 매칭.
2. **Action Week 삽입** — 이론 멘토링 3회 연속을 방지하고, 중간에 현장 실행 주간을 넣어 "진짜 달라진 게 있는가" 를 확인. 임직원 몰입도를 유지하고 창업가는 실제 적용 경험을 얻음.
3. **성과 데이터 리포팅** — 단순 참여 사진이 아니라, 참여자 사전·사후 진단 점수 · 창업가 KPI 변화 · 정성 인터뷰를 묶어 **코오롱 지속가능경영보고서에 바로 쓸 수 있는 포맷** 으로 리포트 제공.

## 5. 언더독스 자체 도구·네트워크가 어떻게 녹았나

- **ACT-PRENEURSHIP**: 임직원 멘토 사전 진단 (전문성 영역 매핑용) 과 창업가 사전·사후 진단 양쪽에 활용.
- **IMPACT 창업방법론**: 창업가 코칭의 기본 프레임. 6단계 중 참여 창업가의 현재 단계에 맞는 모듈만 선별.
- **DOGS Team Building**: 매칭된 페어가 한 팀처럼 움직여야 하므로 초반 1회 투입 (DISC 기반 24문항).
- **4중 지원 체계**: 코오롱 임직원 멘토 (전문 멘토단) + 외부 컨설턴트 풀 + 언더독스 전담 코치 + 동료 (타 매칭 페어) 네트워크. 단일 멘토 구조를 피함.
- **ESG 측정 1,600개 기업 노하우**: 코오롱 지속가능경영보고서 연계 리포팅 포맷에 활용.

## 6. 신입 PM 이 이 케이스에서 배울 점

이 케이스의 **진짜 교훈** 은 사업 내용보다 작성 과정에 있습니다.

**(1) Value Chain 없이 장표부터 쓴 2주의 교훈**. 초반 PM 은 RFP 를 받고 바로 PPT 섹션을 채우기 시작했습니다. 섹션마다 "우리 역량 자랑 + 과업 나열" 로 빽빽하게 채웠지만, 대표·본부장 피드백 때마다 논리 구조가 바뀌어 섹션 순서를 뒤집는 일이 반복. **2주 동안 무한 수정**. 최종본이 나와도 "그래서 이 프로보노가 어떤 임팩트를 만드는가" 가 한 줄로 안 잡혔습니다.

**(2) 반나절 리셋의 효과**. 중간에 팀이 모두 멈추고 A4 한 장에 5계층 (Input → Activity → Output → Outcome → Impact) 을 그렸습니다. 그 뒤:
- 각 계층별 핵심 숫자·도구·증거를 채움 (반나절)
- 이 한 장을 팀 전원이 공유 (공통 언어 확보)
- 섹션별 작성은 **1.5일** 만에 완성
- 피드백은 "이 Outcome 수치 근거 보강" 수준의 로컬 수정으로 수렴
- 최종본 품질은 이전 2주치 버전보다 뚜렷이 나음

**(3) 숫자로 본 교훈**. 2주 무한 수정 → Value Chain 반나절 + 작성 1.5일 = 약 2일. **8배 단축** 이 아니라, 2주 동안 만든 내용 자체가 사용 가능한 자료가 되지 못했기에 **사실상 모든 시간 절약**.

**(4) 프로세스로 체화하기**. 이 교훈을 체화하려면 두 가지 습관:
- **RFP 받고 첫날 ~ 이튿날**: 담당자 통화 + RFP 정밀 분석 + Value Chain 5계층 1페이지 ([Ch.3](../../02-field/03-first-24h.md), [Ch.4](../../02-field/04-impact-value-chain.md))
- **초안 쓰기 전 팀 공유**: Value Chain 한 장을 대표·본부장·PM·서포터가 같이 보며 "이게 말이 되는가" 합의. 합의 후에만 섹션 작성 시작.

**(5) 프로보노 사업의 특수성**. 일반 창업교육과 달리 **B2B 기업 임직원의 경험 설계** 가 한 축. 임직원이 "의미 있었다" 고 느끼도록:
- 참여 시간 대비 임팩트를 구체 수치로 보여주는 리포트
- 임직원 본인의 전문성이 실제로 기여했다는 증빙
- 그룹 내 CSR 발표 자리에 쓸 수 있는 스토리텔링 자료

이 세 가지를 제안서 Section V 에 명시하면 B2B 기업 평가에서 점수 차이가 납니다.

**(6) 주의**. "프로보노" 는 대외 홍보 가치가 커서 기업이 **이벤트성** 으로 끝내기 쉽습니다. 제안사로서 이벤트 프레임을 경계하고, **측정·개선·재참여** 의 사이클로 설계해야 다음 해 재계약·2기로 연결됩니다.
